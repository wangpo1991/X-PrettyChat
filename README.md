# X-PrettyChat
Netty、Chat、Spring  Learning

将Netty整合到实际项目中的一次实践，整个项目将覆盖到网关层、聊天服务、认证服务等等

项目初始化
   
        X-Gateway      网关
        X-Chat-Server  聊天业务
        X-Auth-Server  认证业务
        X-Client       网关下游服务器客户端
        X-Message      消息层
        X-Commons      公用组件
        X-Register     注册中心
